# ARI Obsidian Sync

Keep your Obsidian vault in sync with your agent's knowledge base. Trigger a full vault sync to ensure all notes, decisions, and context are up to date.

## Features
- One-click vault sync
- Real-time knowledge alignment
- Error reporting
- Works with local vaults

## Usage
Say any of these phrases:
- "Sync Obsidian"
- "Update notes"
- "Pull vault"
- "Push changes"
- "Check Obsidian"

The agent will run the sync script and report status.

## Requirements
- Local Obsidian vault at `~/Documents/Main (ARI)/`
- `obsidian-watcher.sh` script available
- Write access to vault directory
