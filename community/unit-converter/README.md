## What It Does
Voice-first unit conversion for cooking, travel, and everyday questions. Ask in natural language, get instant answers.

## Suggested Trigger Words
- "convert"
- "unit converter"
- "convert units"
- "how many"
- "conversion"

## Setup
- No API keys required.

## How It Works
1. Trigger the ability with a conversion request
2. The LLM parses your natural language and does the math
3. You get a spoken answer in 1â€“2 sentences
4. Ask more conversions or say "done" to exit
