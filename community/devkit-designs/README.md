# OpenHome Devkit Designs

3D print files for the OpenHome Devkit V1.0 enclosure. Print your own or use as a base for custom designs.

## Files

- `Devkit Print.3mf` — full print-ready project (recommended)
- `STL Files/` — individual parts for slicing
- `Step Files/` — STEP files for editing in Fusion 360, FreeCAD, SolidWorks, etc.

## Parts

| Part | Description |
|------|-------------|
| Front Cover | Front face of the enclosure |
| Main Body | Main enclosure body |
| Mounting Bosses | Internal mounting points |

Have a remix? Open a PR and add your design here.
