# ARI Council Deliberation

A voice-activated AI council that deliberates on questions using 7 weighted philosophical advisors. ARI (Autonomous Rare Intelligence) uses a council-weighted system where each seat brings a distinct philosophical perspective.

## Council Seats

| Seat | Weight | Focus |
|------|--------|-------|
| Aaron Swartz | 0.25 | Information freedom, open access |
| Vitalik Buterin | 0.20 | Decentralization, mechanism design |
| Hal Finney | 0.15 | Cryptographic privacy, digital cash |
| Buckminster Fuller | 0.15 | Systems thinking, doing more with less |
| Aldous Huxley | 0.10 | Consciousness, perception |
| Satoshi Nakamoto | 0.10 | Trustless systems, sovereign money |
| The Stranger | 0.05 | Contrarian perspective |

## Trigger Words

- "Ask the council"
- "Council deliberation"
- "What does ARI think"
- "Ask ARI"
- "Deliberate on..."
- "Philosophical question"

## How It Works

1. Say a trigger phrase to activate the council
2. State your question
3. The council deliberates using the device's LLM (or OpenClaw gateway if available locally)
4. Each seat weighs in based on their philosophical alignment
5. A weighted consensus is synthesized and spoken
6. You can ask a follow-up question or adjourn

## OpenClaw Integration

If you have [OpenClaw](https://github.com/anthropics/claude-code) running locally on port 18789, the ability will route questions through the full ARI council system for richer deliberations. Without OpenClaw, it simulates the council using the device's built-in LLM.
